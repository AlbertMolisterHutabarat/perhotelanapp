# Sistem Informasi Perhotelan (SIP)
## User
    * Resepsionis
    * Pelanggan
## Web
    Resepsionis :
        * Menerima reservasi kamar
        * Memberikan informasi kamar yang tersedia
        * Melayani keluhan dari pelanggan
        * Mengolah data kamar           
        * Fasilitas 
        * Cek in & Cek Out
        * Kebijakan & Aturan Hotel
## Mobile
    Pelanggan :
        * Pemesanan Kamar & Tamu
        * Cek Harga kamar
        * Cek Stok Kamar yang tersedia
